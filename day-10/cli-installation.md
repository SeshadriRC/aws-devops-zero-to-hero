### Cli-installation on Ubuntu

[doc](https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html)

```bash
$ curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
unzip awscliv2.zip
sudo ./aws/install

```
